# Dynamic Slideshow

## Usage

```js
const $img = [
  {
    src: "img1.jpg",
    caption: "Image Caption 1"
  },
  {
    src: "img2.jpg",
    caption: "Image Caption 2"
  },
  {
    src: "img3.jpg",
    caption: "Image Caption 3"
  },
  {
    src: "img4.jpg",
    caption: "Image Caption 4"
  }
];

const slideshow = new Slideshow($img, ".slideshow");
```
